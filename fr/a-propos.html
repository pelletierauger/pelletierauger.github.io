Je suis un artiste et un programmeur Portrait par
Sophie Yanow vivant à Tiohtià:ke (Montréal, Québec, Canada). Après une enfance passée tout près de la rivière Harricana à Amos, en Abitibi, et une adolescence aux berges du Richelieu à Saint-Ours, en Montérégie, je suis arrivé à Montréal pour faire des études en cinéma d’animation au Cégep du Vieux Montréal. J’ai produit et réalisé plusieurs courts métrages d’animation qui ont fait la tournée des festivals, de Beijing à Žilina, en passant par Bergame, Boston, Brisbane, Čačak, Kanazawa, Kochi, Melbourne, Nairobi, Ottawa, Paris, Rio de Janeiro, Sapporo, Singapour, Tāmaki Makaurau, Tokyo, Toronto, Třeboň, Whitehorse, Wrocław et ailleurs.

Je suis également impliqué dans le milieu montréalais de la bande dessinée indépendante. Mon livre Mélasse est publié chez l’Oie de Cravan et a été nominé aux prix Expozine de la presse indépendante. Mes dessins et mes bandes dessinées se trouvent dans des publications telles que Liberté, L’actualité, Planches, et l’édition collective du livre Ping Pong de Zviane publié chez Pow Pow.

En 2016, je suis retombé amoureux des mathématiques et de la programmation, choses qui dormaient en moi depuis les innombrables moments de plaisir que j’avais passé en apprenant à programmer avec HyperCard à l’âge de 15 ans. Les outils de programmation créative tels que p5.js et SuperCollider me permettent aujourd’hui d’unir ma passion pour les arts visuels avec celle pour l’écriture d’algorithmes et d’objets mathématiques. Ce site Web est dédié à la présentation de ces expérimentations.

Inspiré par les valeurs véhiculées par le mouvement du logiciel libre, je partage librement tout mon travail de programmation. À moins d’une indication contraire, tout ce que vous voyez sur ce site peut être réutilisé, modifié et redistribué, dans le respect de certaines conditions décrites dans les licences utilisées.

Ce site Web

Pour réaliser ce site Web Portrait par
Maxime Gérin, j’ai développé une application Node.js qui lit une base de données puis génère automatiquement tous les fichiers html. J’ai créé cette application parce que je cherchais un « générateur de site statique » (static site generator) à la fois simple et offrant quelques fonctionnalités qui me sont importantes (tel que la possibilité de gérer une base de données de contenu bilingue), et j’étais insatisfait des générateurs déjà existants. Le code source de cette application est sur GitHub (avec le code source de ce site Web au complet). Préparez-vous à
une absence complète
de documentation.Le module principal est maker.js. Il lit les données présentes dans les dossiers pages et blog, et utilise des modules supplémentaires contenus dans les dossiers formatters et node_modules. Les formules mathématiques affichées sur le site ont été mises en forme avec le module Mathjax-Node et la syntaxe des boîtes de code informatique a été colorée avec le module Highlight.js ainsi qu’avec un peu de code supplémentaire.

La police de caractères Crédits
typographiques utilisée pour le corps de ce site est Eb Garamond, une recréation des fontes originales de Claude Garamont telles qu’utilisées dans un spécimen imprimé en 1592 par Konrad Berner. Eb Garamond a été créée par Georg Duffner et est distribuée librement avec une licence Open Font sil. La police utilisée pour l’affichage de code informatique est Inconsolata, créée par Raph Levien et aussi distribuée avec une licence Open Font sil. La police pixelisée utilisée ça et là est Silencieux, une version modifiée de Silentype, une police créée par Sean McNamara. J’ai développé Silencieux pour ajouter les signes diacritiques français à Silentype. Ces deux polices sont aussi distribuées librement avec la même licence.

Le code css des notes de marge a été adapté depuis Tufte css, un projet créé par Dave Liepmann d’après les travaux d’Edward Tufte. Tufte css est distribué librement avec une licence mit.
